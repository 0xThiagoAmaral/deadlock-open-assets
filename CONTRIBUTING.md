# Contributing

Thank you for helping build the Deadlock Open Assets Hub.

## What we need

- Updated extractions after game patches
- Missing hero/item icon mappings
- Codename corrections (VPK internal names)
- Documentation improvements
- Bug reports in extraction pipeline

## Update workflow

```powershell
# 1. Re-extract from Steam (after a patch)
python tools/extract_deadlock_assets.py --tier complete

# 2. Rebuild manifests
python tools/build_community_hub.py --sync-remote

# 3. Verify
python -c "import json; h=json.load(open('manifests/hub.json')); print(h['stats'])"

# 4. Package large assets for release (optional)
python tools/package_release.py
```

## Pull request checklist

- [ ] `manifests/hub.json` regenerated
- [ ] `docs/CATALOG.md` updated (auto-generated by build script)
- [ ] No API keys or Steam paths in committed files
- [ ] No files from `.gitignore` (cheat scripts, secrets, CLI binary)
- [ ] Patch date noted in PR description

## What NOT to submit

- `deepseek.txt`, API keys
- `super_team_umbrella/`, `deadlock_scripts/` (internal tooling)
- `tools/Source2Viewer-CLI/` (auto-downloaded)
- Full particle tarball in git (use GitHub Releases)

## Code style

- Python 3.10+
- Match existing `tools/` conventions
- ASCII-only in `.bat` files

## Questions

Open a GitHub issue with the patch version and what assets seem missing.
